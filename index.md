# Das ultimative Markdown & MyST Handbuch

Willkommen in deinem neuen Jupyter Book! Dieses Buch dient als umfassende Referenz und Demonstration darüber, was heutzutage alles mit modernem Markdown, der **MyST (Markedly Structured Text)** Syntax und Jupyter Book 2 möglich ist.

Markdown ist längst nicht mehr nur eine einfache Auszeichnungssprache für fetten oder kursiven Text. In Kombination mit dem MyST-Ökosystem verwandelt es sich in ein mächtiges Werkzeug für technisches Schreiben, wissenschaftliche Publikationen und interaktive Dokumentationen.

## Inhaltsverzeichnis des Handbuchs

In den folgenden Kapiteln wirst du lernen, wie du Dokumente perfekt strukturierst, komplexe Medien einbindest und die volle Power von Jupyter-Code-Zellen nutzt:

1. **{doc}`01_basics`**: Die Fundamente von Markdown (Überschriften, Listen, Tabellen, Formatierungen).
2. **{doc}`02_advanced_formatting`**: Fortgeschrittene Elemente wie mathematische Formeln, Diagramme und Zitate.
3. **{doc}`03_myst_features`**: Spezifische MyST-Features wie Infoboxen (Admonitions), Tabs und Seitenelemente.
4. **{doc}`04_code_interactivity`**: Die Integration von ausführbarem Code (Python) und interaktiven Visualisierungen direkt im Text.
5. **{doc}`05_simple_notebook`**: Test für einen Code
